# idm364-tc3363

This project is a dynamic E-commerce web application that sells Dim Sum. 

## Planning

- Home
    - Header
        - Top left: icon + logo
        - Top right: account + cart
        - Background: banner + headings
    - Subheader: categories of items
    - Cards of dishes
        - Image of the dish
        - English name (primary)
        - Cantonese name (secondary)
        - Description (tertiary)
        - Price/serve
        - Quantity swatches
- Cart
    - Header
    - "Remove All" button
    - Cards with a "Remove" button but no descriptions
    - eBill section
        - Subtotal
        - Tax
        - Total
- Account
    - Selector between "Login" and "Register"
    - Username with "Forgot Username"
    - Email (for registering only)
    - Password with "Forgot Password"
    - "Remember Me" checkbox
    - Confirm button

## Content 

[Menu](/media/menu_v2.json) and images are publically accessible on [Wikipedia](https://en.wikipedia.org/wiki/Dim_sum), prices and descriptions are generated by ChatGPT.
